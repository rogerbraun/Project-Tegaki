Tegaki Project

Tegaki is an ongoing project which aims to develop a free and open-source modern implementation of handwriting recognition software, specifically designed for Chinese (simplified and traditional) and Japanese, and that is suitable for both the desktop and mobile devices.

Screencast video: ogg or youtube.

Packages

Libraries

  • tegaki-python: base Python library (0.2).
  • tegaki-pygtk: base user interface library (0.2).

Applications

  • scim-tegaki: Tegaki integration in SCIM (0.1).
  • tegaki-recognize: Tegaki integration in the desktop (0.2).
  • tegaki-train: Train Tegaki with your own handwriting (0.2).

Models

  • tegaki-zinnia-japanese: Japanese model for the zinnia recognizer (0.2).
  • tegaki-zinnia-simplified-chinese: Simplified Chinese model for the zinnia recognizer (0.2).

Developer tools

  • tegaki-tools: tools to convert data, train models and evaluate models (0.2).

( ) indicates the latest version.

Engines

Tegaki is designed to be able to use several recognition engines. So far however, we only support Zinnia.

One of the ongoing goals of the project is to implement fast and reliable recognition engines using state-of-the-art techniques.

Models

In addition to Zinnia, you need to install at least one of the following models:
  • tegaki-zinnia-simplified-chinese
  • tegaki-zinnia-japanese

Models are based on character templates from the Tomoe project.

Development

The source code is managed with git. You can get the latest source tree with the following command:

$ git clone http://www.mblondel.org/code/hwr.git

A web interface is available here.

You can report bugs in our bug tracker.
